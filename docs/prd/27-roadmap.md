# PRD 27 — Roadmap (Eight Phases)

> **Source:** PA 27, lines 11090–11571.

## Area overview

The end-to-end delivery plan, expressed as eight phases. Each phase has a coherent theme and exits when its outcome is shipped.

## Phase 1 — Core Wiki Foundation

Pages, spaces, editor, comments, real-time collaboration, basic search, page history, file uploads, basic permissions. Establish the **wiki baseline** parity with Confluence/Notion.

**Status:** Shipped.

## Phase 2 — Collaboration & Content Quality

Templates, viewer comments, comment resolution, page tree polish, page metadata, owner assignment, page favorites, attachment indexing, version history.

**Status:** Largely shipped (Business+).

## Phase 3 — Business & Enterprise Security

SSO (Google + SAML + OIDC + LDAP), MFA, API keys, sharing controls, security settings, allowed domains, page-level permissions, audit logs, retention controls.

**Status:** Shipped.

## Phase 4 — AI Knowledge Layer

Editor AI (Ask AI), AI Search with citations, AI Chat with tool calling, MCP endpoint, AI governance.

**Status:** Shipped (Enterprise).

## Phase 5 — Migration & Integrations

Notion, Confluence, DOCX import; PDF export; productivity integrations (Slack/Teams notifications, webhooks, GitHub).

**Status:** Imports shipped. Notifications + integrations partial / planned.

## Phase 6 — Governance & Intelligence

Page verification (expiring + QMS), documentation health center, knowledge gap detection, broken-link detection, duplicate-content detection, expert insights.

**Status:** Verification shipped. Health center + gap detection partial / planned.

## Phase 7 — External Knowledge Experience

Public docs portals, custom domains, branding removal, guest access, public feedback forms.

**Status:** Public sharing shipped. Doc portals partial.

## Phase 8 — Enterprise Operations & Scale

SCIM provisioning, air-gapped deployment, advanced AI governance (sensitive-space exclusion), multi-region (planned), federated workspaces (planned).

**Status:** SCIM + air-gapped shipped. Federation roadmap.
